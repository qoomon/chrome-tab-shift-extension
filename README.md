# chrome-tab-shift-extension
This extension adds shortcuts to shift tabs.

#### Shortcuts
chrome://extensions/shortcuts
* Shift Left: 
  * **Ctrl ⇧ ←** 
  * `MacOS` **⌃ ⌘ ←**
* Shift Right: 
  * **Ctrl ⇧ →** 
  * `MacOS` **⌃ ⌘ →**
* Shift to Left Edge:
  *  **Ctrl ⇧ ↑** 
  * `MacOS` **⌃ ⌘ ↑**
* Shift to Right Edge:
  *  **Ctrl ⇧ ↓** 
  * `MacOS` **⌃ ⌘ ↓**
